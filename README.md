# スタック ＆ キュー 可視化アプリケーション (Stack & Queue Visualizer)

Rubyの `tk` ライブラリを使用した、データ構造の「スタック（Stack）」と「キュー（Queue）」の挙動を視覚的に理解するためのデスクトップアプリケーションです。

データの挿入や取り出しを行った際に、どの要素が操作対象になるかが色とアニメーションで直感的に把握できます。

## 特徴

- **スタックの可視化 (LIFO: 後入れ先出し)**: 末尾に追加され、末尾から取り出される挙動を視覚化します。
- **キューの可視化 (FIFO: 先入れ先出し)**: 末尾に追加され、先頭から取り出される挙動を視覚化します。
- **アニメーション演出**: 取り出し（Pop / Dequeue）ボタンを押した際、対象となる要素が一瞬ハイライトされてから消去されます。
- **自動インクリメント機能**: 値を入力して追加ボタンを押すと、入力欄の文字が自動的に次の文字（`A` → `B`、`1` → `2` など）に進むため、連続したテストデータの投入が容易です。

## 動作環境

- Ruby 2.4 以降（Tkライブラリが動作する環境）
- Tk (Tcl/Tk) 

## インストールと準備

Ruby 2.4以降では `tk` ライブラリが標準添付から外れているため、事前に Gem のインストールが必要です。また、システムに Tcl/Tk がインストールされている必要があります。

### 1. 依存Gemのインストール
端末（ターミナル / コマンドプロンプト）で以下のコマンドを実行してください。

```bash
gem install tk
※ Windows環境で「Ruby+Devkit」を使用している場合や、Mac環境で Homebrew を使用している場合は、必要に応じて tcl-tk のシステムインストールを行ってください。

使い方
本リポジトリのスクリプト（例: vis_stack_queue.rb）を実行します。

Bash
ruby vis_stack_queue.rb
画面上部の「値入力」欄に好きな文字や数字を入力します（デフォルトで A が入力されています）。

各種ボタンを押して、データの動きを確認します。

Stack Push: スタックの末尾に要素を追加します。

Stack Pop: スタックの最後に追加された要素（右端）をハイライトして取り出します。

Queue Enqueue: キューの末尾に要素を追加します。

Queue Dequeue: キューの最初に追加された要素（左端）をハイライトして取り出します。

画面イメージ（構造）
+-------------------------------------------------------------+
| 値入力: [ A ]  [Stack Push] [Stack Pop] | [Queue Enqueue] [Queue Dequeue] |
+-------------------------------------------------------------+
|                                                             |
|  スタック (Stack)                                           |
|  [ A ] [ B ] [ C ] <-- Push / Pop (末尾)                     |
|                                                             |
|  キュー (Queue)                                             |
|  <-- Dequeue (先頭) [ A ] [ B ] [ C ] <-- Enqueue (末尾)    |
|                                                             |
+-------------------------------------------------------------+
